## food-sales-data-analysis  
本目录下是对数据进行分析和预测的代码实现  
## 编程语言及运行环境  
--Python  
## 目录说明  
--data2目录存放原始数据（数据较大，已压缩，请先解压）  
--result目录存放预测结果（数据较大，已压缩，请先解压）
